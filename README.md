# Regional Sales Analytics & Revenue Optimization

Exploratory data analysis and revenue optimization insights derived from
7,560 sales transactions using Python.

## Problem Statement
Retail businesses struggle to identify which regions, customer segments,
and time periods drive the most revenue. This project uncovers those
patterns and recommends actionable strategies.

## Key Findings
- Top revenue-generating states contribute **70.35%** of total revenue
- Evening flash sales and December campaigns identified as highest ROI opportunities
- Customer segmentation revealed distinct high-value demographic clusters
